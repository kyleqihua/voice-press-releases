# Voice Press Releases

This repository hosts releases for [Voice Press](https://github.com/kyleqihua/p4_voice2word), a macOS voice-to-text transcription app.

## Download

Check the [Releases](https://github.com/kyleqihua/voice-press-releases/releases) page for the latest version.

## Auto-Update

Voice Press includes automatic update checking via Sparkle. The app will notify you when a new version is available.
